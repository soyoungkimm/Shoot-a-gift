<!-- ABOUT THE PROJECT -->
## 프로젝트 소개 :smile:

프로젝트명 : Shoot a gift<br>
제작 기간 : 12월 4일 ~ 12월 7일<br>
Unity 2020.3.9 버전 이용<br>
Oculus 사의 ‘Quest 2’라는 VR기기를 사용하여 만든 게임<br>
양손을 이용한 플레이 가능<br>
Oculus Air link를 통해 데스크톱 Unity와 연결하여 개발함<br>


<br><br>
<!-- GETTING STARTED -->
## 구현 기능 :pencil2:

-  헤드 상하좌우로 돌아보기 기능
- 양손 애니메이션
- 오브젝트 grab 기능
- 플레이어 이동, 회전
- 플레이어 중력
- 플레이어 앉기
- 게임 시작 버튼 (직접 가서 눌러야 함)
- 오브젝트 랜덤 드랍
- 게임 내 ui
- 총 발사
- 타이머
- score 상승
- total score 출력
- close 버튼 (직접 가서 눌러야 함)
